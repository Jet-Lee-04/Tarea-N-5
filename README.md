# Calculadora Básica en Python

Desarrollé una calculadora simple que funciona desde la línea de comandos utilizando Python.

## Descripción

Mi calculadora permite realizar operaciones matemáticas básicas como:

- Suma  
- Resta  
- Multiplicación  
- División  

## Cómo usar

1. Ejecuto el archivo `calculadora.py`  
2. Selecciono la operación que quiero realizar (opciones del 1 al 5)  
3. Ingreso los números con los que quiero trabajar  
4. Veo el resultado directamente en consola
